# yadi
"Yet another DSS interface" (yadi) is a high-level interface for getting things done with the EPRI OpenDSS Electric Distribution System Simulator in Python.

## Features
yadi makes integrating OpenDSS into your research and power system studies seamless. It provides support for several key features.

### Model-based analysis
1. Circuit data analysis
2. Time series analysis
3. Voltage sensitivity analysis
5. Electric vehicle analysis
6. Model-based hosting capacity analysis

### Model-free analysis
1. Advanced metering infrastructure (AMI) dataset generation
2. Regression-based sensitivity analysis
3. Model-free hosting-capacity analysis
